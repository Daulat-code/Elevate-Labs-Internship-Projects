A Spring Boot RESTful API for managing books and authors in a bookstore.
Supports CRUD operations, pagination, sorting, and dynamic filtering with JPA Specifications.
Includes DTO mapping, Swagger/OpenAPI docs, and a clean layered architecture.

#Features

1. Manage Books (Add, Update, Delete, Get)

2. Manage Authors

3. Search & filter books (by genre, price, author)

4. Pagination & Sorting support

5. Layered architecture: Controller → Service → Repository → Entity

6. DTO & Mapper for clean request/response handling

7. Swagger/OpenAPI documentation

#Tech Stack

1. Java 21
2. Spring Boot 3
3. Spring Data JPA (Hibernate)
4. MySQL
5. Swagger / OpenAPI
6. Maven
